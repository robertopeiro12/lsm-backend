# LSM Backend API

Backend para la aplicación de aprendizaje de Lengua de Señas Mexicana (LSM).

## Tecnologías

- FastAPI
- MySQL
- Firebase Authentication

## Endpoints

- `/docs` - Documentación Swagger
- `/redoc` - Documentación alternativa

## Variables de entorno necesarias

- `DB_HOST`
- `DB_USER`
- `DB_PASS`
- `DB_NAME`
